# AI Content Marketing using UGC Ads

This project is about using AI to create User Generated Content (UGC) ads for a SaaS product using prompt engineering techniques.

Product: HireBoost AI (AI Resume Builder for students and freshers)

Problem:
Students struggle to create ATS-friendly resumes and fail to get interview calls.

Tools Used:
- ChatGPT
- Gemini
- Claude
- Lovable (for landing page)

What I Built:
- UGC ad hooks
- Instagram Reel scripts
- CTA variations
- AI prompt system for marketing content

Sample Output:

Hooks:
- This AI fixed my resume in 5 minutes
- I started getting interview calls after this tool

CTA:
- Try it free today
- Build your resume smarter

This project demonstrates how AI can be used in real-world content marketing and advertising.
